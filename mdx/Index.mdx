# 示例

React Antd Extend Library 是一个用于二次封装 [Ant Design](https://ant.design/) 组件以适合自身业务需求的组件的模版。

## 使用

```jsx harmony
import { MyInput } from 'react-antd-extend-library';

const Demo = () => (
  <MyInput
    label="用户名"
  />
);

export default Demo;
```
